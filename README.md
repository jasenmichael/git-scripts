# git-scripts

```bash
curl -fsSL https://raw.githubusercontent.com/jasenmichael/git-scripts/main/git-scripts | bash
```

or

```bash
git-scripts list
git-scripts [script]
```

```bash
git-scripts install-cursor

git-scripts install-deskflow

git-scripts install-docker

git-scripts install-ghostty

git-scripts install-rust

git-scripts install-unruntime

git-scripts install-windsurf
```
